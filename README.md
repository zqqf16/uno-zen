## Installation

Clone this repository to your blog directory.

``` bash
git clone https://github.com/zqqf16/uno-zen.git
```

Move assets out.

``` bash
mv uno-zen/assets ./
```

Edit your configuration file, set theme to "uno-zen"

``` yaml
theme: uno-zen
```

Rebuild

``` 
peanut ./
```
